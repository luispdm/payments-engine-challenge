# payments-engine-challenge

Rust solution to the payments engine challenge.

## Spec

Implementation details at `~/payments-engine-challenge-docs/challenge.pdf`. Read it before changing logic.

## Tasks

Implementation broken into ordered tasks at `~/payments-engine-challenge-docs/NN-name.md`. Pick up the lowest-numbered unfinished task. Tasks reference decisions taken during planning; do not contradict them without a follow-up grill.

## Confidentiality

Per the challenge PDF, no derivative of the PDF may be committed to this repo. The `~/payments-engine-challenge-docs/` directory is outside the repo and never gets committed. Do not paste, quote, paraphrase, or summarize the spec into code, comments, commits, docs, issues, or tests. Refer to the PDF by path only.

## Toolchain
